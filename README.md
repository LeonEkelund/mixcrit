MixCrit is an AI-powered tool that gives producers instant feedback on their mixes. Upload a track and receive actionable insights on balance, dynamics, stereo image, and overall mix quality.

🌐 Live Demo: https://mixcrit.com

## Features

- AI-powered mix analysis
- Fast audio processing
- Structured, actionable feedback
- Privacy-friendly workflow

## Built With

- Next.js
- TypeScript
- MongoDB
- Groq
- Web Audio API
